# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'PxGet'
Import-Module -Name 'PxGet'
```

To install privately:

```powershell
Save-Module -Name 'PxGet' -Path '.'
Import-Module -Name '.\PxGet'
```
